## Installation

1. local 환경에 레파지토리를 clone 해주세요
```
$ git clone https://github.com/GyuhoLee/MPP.git
```
2. 프로그램 실행을 위해 Android Studio가 필요합니다.

3.Android Studio가 설치되었으면 src 폴더를 실행해서, app로 변환하시면 됩니다.
